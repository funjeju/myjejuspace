# 1차 유입 전략 — CCTV
# CCTV 전략

## 목적
CCTV를 목적으로 진입한 사용자를 탐험으로 자연 전환

## 구현 (FEATURE_SPEC.md 참조)
- 지도 위 📹 아이콘으로 CCTV 위치 표시
- 클릭 시 하단 시트로 실시간 영상 오버레이
- 지도는 그대로 유지 (전환 없음)
- 영상 주변 스팟/공간 자동 노출

## 전략적 가치
- "지금 제주 날씨/혼잡도 확인" 목적으로 첫 진입
- 영상 보는 중 주변 맛집/유저공간 자연 노출
- 별도 메뉴 없이 지도 위 오브젝트로 존재
- DAU 상승의 핵심 훅

## 데이터 소스
- 제주 교통정보 시스템 (도로 CCTV)
- 제주관광공사 (관광지 웹캠)
- 국가교통정보센터 ITS API
- 민간 제휴 (카페/식당 등 별도 협약)
